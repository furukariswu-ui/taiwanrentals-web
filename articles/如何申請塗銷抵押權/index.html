如何申請塗銷抵押權

發布日期：2024-07-13

🔸什麼是塗銷抵押權？

申請房貸時，用於擔保的房子必須設定抵押權，以保障銀行在借款人無法還款時可收回房屋。抵押權會在房貸清償後塗銷，房屋才真正屬於您。本文將詳細介紹如何塗銷抵押權以及需要準備的資料。

🔸什麼時候可以塗銷抵押權？

已設定抵押權的房屋，因「權利拋棄、期限結束、債務清償、法院判決」等原因，可以申請塗銷抵押權。最常見的情況是屋主已經清償所有債務，無需再以房子擔保。完成塗銷後，房屋謄本的他項權利部會回復原樣，避免房子被法拍的風險

🔸如何申請塗銷抵押權？

  1. 臨櫃申請
    申請人和債權人準備好所有資料，前往地政事務所當場辦理，大約1小時內完成。如果申請人無法出席，可以委託代理人代辦。
  2. 郵寄申請
    申請人將所有資料郵寄到地政事務所，然後按照約定時間到場辦理。
  3. 線上申請
    如果請代書或律師代為申請，可連上內政部數位櫃台系統，以自然人憑證或工商憑證登入，填寫申請書和資料。申請人檢查並簽署電子簽章，代書或律師完成送件，其他紙本附件則郵寄到地政事務所。

🔸銀行貸款 塗銷抵押權所要哪些文件

  1. 抵押權塗銷同意書
  2. 他項權利證明書
  3. 原設定契約書

🔸私人借貸 塗銷抵押權所要哪些文件

  1. 抵押權塗銷同意書
  2. 債務清償證明書
  3. 他項權利證明書
  4. 債權人身分證明文件
  5. 債權人印鑑證明

地政事務所辦理文件

  1. 登記申請書
  2. 申請人身分證正本和印章
  3. 自然人為身分證或戶口名簿影本
  4. 法人為登記證明及代表人資格證明
  5. 公司為設立登記表或抄錄本正、影本

其他文件

  1. 院判決確定證明文件
  2. 地政機關要求的證明文件

🔸他項權利證明書遺失怎麼辦？

如果遺失他項權利證明書，可以附上「他項權利人切結書」，並在清償證明書中說明文件已經交出並蓋章。如果遺失債務清償證明書，可以向原貸款機構申請新的證明，攜帶身分證和印章至現場辦理即可，通常需2-3個工作天。

🔸未塗銷抵押權的後果

如果清償房貸後未及時塗銷抵押權，可能影響未來的貸款申請或房屋出售。如果仍能找到債權人，可以協商辦理。如果找不到債權人，可透過司法途徑解決問題，取得法院判決證明文件，作為地政事務所接受的依據。

🔸繳完房貸需要馬上塗銷抵押權嗎？

取得債務清償證明書後，是否馬上塗銷抵押權取決於個人需求。如果未來可能需要資金，可以不塗銷，以節省設定規費並增加房屋保障。如果沒有其他計畫，及早塗銷抵押權較為安全。

🔸塗銷抵押權的流程

✓ 收件：核對申請人身分、證明文件。
✓ 核定準登：於土地登記系統查詢輸入準登。
✓ 登錄：逐一登錄核定登記事項。
✓ 校對：檢查資料是否正確。
✓ 結案：於登記案管系統中結案。
✓ 異動歸檔：點收檔案並簽註日期。

塗銷抵押權是房屋貸款流程中的重要一環，確保房屋真正屬於您。了解塗銷流程和所需文件，可以讓您在面對貸款和房屋買賣時更加從容。如果有任何疑問或需要專業幫助，隨時尋求專業諮詢，確保您的權益。
